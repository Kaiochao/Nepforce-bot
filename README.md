## Contributing
contributing.
- Fork the repository
- Clone the just forked repository
- Set the repository upstream
- Create discord app
- Set permission for bot
- Create .secret in root and paste token into file
- ```$ npm i```
- ```$ node ./src/bot.js```
- Make pull request once done
